일정

모든 상담은 전문적인 검사를 통해 이루어집니다. 순서는 아래와 같습니다.

1. 척도체크 확인검사
2. 유형 특질 출력
3. 연상기법
4. 전문 상담 :
MBIT(30%) 애니어그램(30%) 게슈탈트치료법 + 실존주의치료법 + 분석심리학 + 수용심리 + 긍정심리 + 변증법적치료법 + 대상관계이론

선택 가능 옵션

1. A코스 가벼운 상담 :

3. C코스 친구 코스 :
2명이서 B코스를 이용할 수 있는 옵션.

4. D코스 연인 코스 :
성격유형 + 궁합 및 애정소통 그리고 갈등에 대한 코칭.
